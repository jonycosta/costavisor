# CostaVisor Platform

This is the monorepo for the CostaVisor platform, integrating Web, Mobile App, and Backend services.

## Project Structure

*   **`web/`**: Next.js application for the public website and booking platform.
*   **`app/`**: React Native (Expo) application for iOS and Android.
*   **`backend/`**: Node.js API server (NestJS/Express) powering the platform.
*   **`docs/`**: Project documentation (Architecture, Design System, Database Schema).
